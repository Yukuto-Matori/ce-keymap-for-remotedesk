# ce-keymap-for-remotedesk

このアプリは Windows11 環境で動作し、キーボードのショートカット機能を追加します。
Android から Parsec でリモート接続した時などで使えないショートカット機能を代替できます。
余計な処理が全く含まれていないので、非常に軽量に動作します。

## 追加されるショートカット機能

キーは変更可能。個別にショートカットの ON/OFF も可能。

- アプリウィンドウ切り替え(`右Alt + 右Shift + A`)
- 拡大率変更 デスクトップ用(`右Alt + 右Shift + D`)
- 拡大率変更 モバイル用(`右Alt + 右Shift + M`)
- Winキー押下(`右Alt + 右Shift + W`)

## 起動＆操作方法

CeKeymap.exe をクリックすれば起動します。
タスクトレイに常駐しているので、右クリックでショートカットキーの変更、個別 ON/OFF の設定変更も
可能。

## 設定一覧

タスクトレイのメニューから以下の変更が行えます。なお、この設定はアプリの場所(exe)と同じフォルダ内に
settings.json 形式で保存されています。他の PC にコピーすることで、設定の共有もできます。

- PC起動後に自動起動 ... レジストリの Startup にこのアプリを登録する/登録解除する
- 機能設定
  - アプリウィンドウ切り替え ... 機能のON/OFF, キーバインド変更 が可能
  - 拡大率変更 デスクトップ用 ... 機能のON/OFF, キーバインド変更, 拡大率の指定 が可能
  - 拡大率変更 モバイル用 ... 機能のON/OFF, キーバインド変更, 拡大率の指定 が可能
  - Winキー押下 ... 機能のON/OFF, キーバインド変更 が可能
- 開発者向け
  - アプリログを開く ... アプリの場所(exe)と同じフォルダ内の log.txt をシステムデフォルトのアプリで開く

## Windows SmartScreen の警告について

このアプリはコード署名を行っていないため、ダウンロードした CeKeymap.exe を初めて実行すると、
Windows SmartScreen が「Windows によって PC が保護されました」という警告を表示することがあります。
これは発行元が未検証の実行ファイル全般に対する一般的な警告であり、このアプリ固有の問題ではありません。

起動するには以下の手順を行ってください。

1. 警告画面内の「詳細情報」をクリックする
2. 表示された「実行」ボタンをクリックする

一度この手順で実行すると、以降は同じ警告は表示されません。
